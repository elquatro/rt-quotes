```yaml
  release: &r0
    id: 0
    date: 27.08.2006
    link: http://www.radio-t.com/p/2006/08/27/podcast-0/
  time: 00:00:00
  text: >
    Сегодня мы представляем вашему драгоценному вниманию пилотный выпуск нового
    подкаста пока условно названый t-radio
```
